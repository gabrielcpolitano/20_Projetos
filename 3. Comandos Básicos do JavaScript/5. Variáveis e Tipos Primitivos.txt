
Tipos de Dados e Variáveis

Tipos de Dados
1. String: Representa uma sequência de caracteres, pode ser escrita entre aspas duplas (" "), aspas simples (' ') ou crase (` `).

   Exemplos:
   "olá"
   'olá'
   `olá`

2. Number: Representa números, incluindo inteiros e decimais.

3. Boolean: Representa valores lógicos, sendo true ou false.

Comentários
- Linha única: Para comentários de uma linha, use // no início.
  // Este é um comentário de uma linha

- Bloco: Para comentários que abrangem várias linhas, utilize /* */.
  /* 
  Este é um comentário
  de múltiplas linhas
  */

Variáveis
- Pode: 
  - Começar com uma letra, $ ou _
  - Conter letras, números, acentos e símbolos (exceto espaços)

- Não Pode: 
  - Começar com números
  - Conter espaços
  - Ser uma palavra reservada (ex.: if, else, for)

Dicas:
- Lembre-se que há diferenciação entre maiúsculas e minúsculas (var e Var são diferentes)
- Use nomes coerentes para suas variáveis, facilitando o entendimento do código
- Evite usar nomes de variáveis como a, b, ou num1, num2, e opte por nomes significativos
